# go-core
游戏发行核心模块

# 使用方式
```
go get github.com/cngamesdk/go-core
```

# 目录

- config 配置文件模型，如常见模型、Sql、Redis、Log模型等
- cryptor 加解密相关
- goroutine 协程相关
- idcard 身份证相关
- log 日志相关
- model 模型相关，如数据库模型、api模型、输入和输出相关
- pay 支付相关，对国内常见支付网关进行封装，如微信、支付宝等
- sms 短信相关
- translator 翻译器相关
- util 常见工具类
- validate 验证器相关
